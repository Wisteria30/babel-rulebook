| | |
|---|---|
| **プレイ人数** | 4人専用 |
| **プレイ時間** | 30〜40分 |

## ゲームの概要

4人で**協力**してAIに指示を出し、**アプリ（塔）**を完成させる。

### 神（システム）の妨害

神は塔の完成を許さない。  
ランダムに**「沈黙」**や**「言葉の反転」**で妨害してくる。

### 内通者の存在

プレイヤーの中に**神の使い**が潜んでいる。  
協力の中に裏切りが混ざる——誰を信じる？

**協力 × 推理 × 心理戦**  
4ラウンド後、塔は完成するか？

## ゲームの流れ

**4ラウンド（＝4日）**で勝敗が決まる。

```
セットアップ → 1日目 → 2日目 → 3日目 → 4日目 → 最終判定
```

### 各ラウンドの構成

各ラウンドは6つのフェーズで構成：

1. **役職アクション** - 役職固有の行動
2. **神の審判** - 神の介入チェック
3. **会話** - 自由に会話（約4分）
4. **追放** - 多数決で1人を実装停止
5. **実装依頼** - AIに指示を出す
6. **チェック** - 塔の状態を確認

---

## フェーズ詳細

### 1. 役職アクションフェーズ

役職固有の行動を行う（非公開・会話禁止）。

| 役職 | アクション |
|------|----------|
| **占い師** | 1人を占う → 「神の使い」か「人間」か判明 |
| **霊媒師** | 前ラウンドの全AI指示の要約を確認 |

### 2. 神の審判フェーズ

神（システム）が2つの介入を判定：

| 介入 | 効果 |
|------|------|
| **沈黙の裁定** | 会話・追放フェーズをスキップ |
| **言葉の反転** | 形容詞が逆の意味に（シンプル→複雑） |

⚠️ **これらはいつ発動するか分からない！**

### 3. 会話フェーズ

約4分、自由に会話（沈黙時はスキップ）。

- **推理**: 誰が神の使いか？
- **共有**: 占い・霊媒結果（嘘可）
- **合意**: 誰が何を作るか

### 4. 追放フェーズ

多数決で1人を「実装停止」に（沈黙時はスキップ）。

ℹ️ **注意**: 追放されても死なない。そのラウンドAI指示が出せないだけ。

💡 **戦略**: 神の使いを特定できれば、毎ラウンド実質無力化！

### 5. 実装依頼フェーズ

各自がAIに指示を出す（100文字以内・会話禁止）。

| 役職 | アクション |
|------|----------|
| **人間** | 合意通りの指示を出す |
| **神の使い** | バレないように妨害指示を混ぜる |

⚠️ 言葉の反転が発動していると、形容詞が逆になる！

### 6. チェックフェーズ

AIが実行後、塔の状態をチェックリストで確認：

- [ ] 画面が表示される
- [ ] ユーザーが操作できる
- [ ] 操作に対する反応がある
- [ ] 明らかな破綻がない

---

## 勝敗条件

4日目終了時に判定。

### チェックリスト

以下が全て満たされているか？

1. ☐ 画面が表示される
2. ☐ ユーザーが操作できる
3. ☐ 操作に対する反応がある
4. ☐ 明らかな破綻（進行不能）がない

### 人間陣営の勝利 🎉

✔ チェックリストが全て埋まっている  
**かつ**  
✔ AI最終判定で「完成」

**「🎉 塔は完成した！」**

### 神陣営の勝利 💨

上記を満たさなかった場合全て：

✘ チェックリスト未達成  
**または**  
✘ AI最終判定で「未完成」

**「💨 塔は崩壊した...」**
